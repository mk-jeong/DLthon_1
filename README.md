# DLthon_1
Repository for 1st DLthon team3

---

## Envirionments
---
- Linux Ubuntu
- CUDA version: 12.1
- Python version: 3.10.20

## Installation
---
```
conda create -p ./venv/dlthon1 python=3.10.20
conda activate ./venv/dlthon1
which pip # pip 설치 위치 확인 ./venv/dlthon1
pip install -r requirements.txt
```
